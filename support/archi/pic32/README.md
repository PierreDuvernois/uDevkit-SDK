# PIC 32 bits support

This architecture support file handle all 32 bits famillies of Microchip. compared to 16 bits, 32 bits range offers various advantages :

* C++ compiler support
* larger memory addressing
* code relocalisation
* execution from RAM

(273 devices on july 2017)
